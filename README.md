A Minecraft plugin I made for the server of my friends and me. At the beginning it was very small and I just had a few ideas. It slowly started to grow.

At first I added basic things like custom join messages and leave messages. I also added a timer to the actionbar. 
Before that, I also tried to copy Hypixel Fireball Bedwars. I played it a lot and I wanted to make something similar. I tried to recreate the feeling of Bedwars and also change and tweak small things so it would work in my own version.

Now Ive created my own clan system where players can join groups and havbe a private chat and custom name colors there.
